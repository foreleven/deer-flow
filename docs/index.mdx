# DeerFlow Documentation

DeerFlow helps you build, run, and operate agent systems with a clear separation between the runtime harness and the application experience built on top of it.

## What is DeerFlow?

DeerFlow consists of two closely related parts:

- **DeerFlow Harness**: the core runtime and SDK layer for building agent systems with memory, tools, skills, sandboxes, and subagents.
- **DeerFlow App**: a reference application built on top of the Harness that shows how to deploy, operate, and use DeerFlow in practice.

This documentation is organized the same way. If you want to understand the core architecture and integration model, start with the Harness. If you want to deploy and use a complete product experience, start with the App.
